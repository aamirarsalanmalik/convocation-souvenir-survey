# Convocation Souvenir Survey (Streamlit)

A sample Streamlit app for collecting student interest in souvenir products and their preferred designs.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud
Push this folder to a GitHub repository, then deploy the repo from Streamlit Community Cloud.

Main app file: `app.py`
